# interview-sim-ml
ML-сервис для проекта "Cимулятор собеседований"

-----------------------------------------------------------------------
Для корректного запуска обработки голосового сообщения (файл voice.py)
необходимо установить некоторые библиотеки:

pip install openai-whisper fastapi uvicorn ffmpeg-python

Запустить командную строку от имени администратора и выполнить:

choco install ffmpeg
